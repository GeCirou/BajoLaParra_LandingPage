# Bajo la Parra - Orfebrería y Cerámica 🌿

Landing page profesional para **Bajo la Parra**, un taller de orfebrería, platería criolla y cerámica de autor liderado por la maestra orfebre Marta Acevedo. Este proyecto busca transmitir la esencia del trabajo artesanal, el detalle y la tradición a través de una experiencia digital fluida y elegante.

## Tecnologías Utilizadas

Este proyecto fue desarrollado con un stack moderno enfocado en la performance y la mantenibilidad:

* **React + Vite**: Para una estructura de componentes robusta y un entorno de desarrollo ultra rápido.
* **Tailwind CSS v4**: Aprovechando las nuevas capacidades "engine-first", configuración mediante variables CSS nativas (`@theme`) y optimización de estilos.
* **Lucide React / Heroicons** (opcional si los usaste): Para la iconografía sutil.
* **Gestos Touch nativos**: Implementación manual de lógica de swipe para mejorar la experiencia en dispositivos móviles.

## Características Principales

* **Diseño Responsive**: Adaptación completa para móviles, tablets y desktops de alta resolución.
* **Optimización de Scroll**: Uso de `scroll-margin-top` para una navegación fluida entre secciones sin solapamiento con el header.
* **Galerías Interactivas**:
    * Carruseles con soporte para teclado (flechas).
    * Navegación táctil (swipe) para una experiencia "app-like" en celulares.
    * Modales de alta resolución para apreciar el detalle de las piezas.
* **Arquitectura de Componentes**: Separación lógica en componentes reutilizables (`Hero`, `ProductGrid`, `Ventas`, `Contacto`).
